# Cursor Button for LocalWP

A LocalWP addon that adds a button to open the current site's public folder in **Cursor**.

## Features

-   **One-Click Access**: Adds an "Open in Cursor" button to the site info panel.
-   **Seamless Integration**: Works directly within the LocalWP interface.
-   **Direct Launch**: Opens the site folder directly in the Cursor app.

## Installation

1.  Clone or download this repository into your Local addons folder:
    -   **macOS**: `~/Library/Application Support/Local/addons`
    -   **Windows**: `%AppData%\Local\addons`
    -   **Linux**: `~/.config/Local/addons`
2.  Restart LocalWP.
3.  Enable the addon from the **Tools > Add-ons** menu if not already enabled.

## Development

1.  Navigate to the folder and install dependencies:
    ```bash
    npm install
    ```
2.  Build the addon:
    ```bash
    npm run build
    ```
3.  To watch for changes and automatically rebuild:
    ```bash
    npm run watch
    ```
